# 🤯 BrainAsm 

A very simple [brainfuck](https://en.wikipedia.org/wiki/Brainfuck) interpreter written in flat assembly language.

## Usage

To compile this code, simply run
```shell
fasm main.asm
```
And to execute run
```shell
./main {filename}
```
